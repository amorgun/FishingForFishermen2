## Usage
1.
```
git clone https://github.com/amorgun/FishingForFishermen2.git
cd FishingForFishermen2
pip install -r requirements.txt
```
2. Put data into `input` folder.
3. Run `run.sh`.
4. The result will be in the file `solution.csv`.
